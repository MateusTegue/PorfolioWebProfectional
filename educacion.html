Formaciones Academicas

A lo largo de mi formación académica, he adquirido una sólida base de conocimientos y habilidades en desarrollo de software y programación. Cada etapa ha sido una oportunidad para aprender, crecer y prepararme para los retos del mundo tecnológico.

Tecnico

Politécnico PIO, Cali (2023-2024) Durante este programa, me he enfocado en el desarrollo de habilidades fundamentales en la programación y la creación de soluciones tecnológicas. A lo largo de mis estudios, he profundizado en conceptos clave como algoritmos, estructuras de datos y programación orientada a objetos, además de adquirir conocimientos esenciales sobre bases de datos relacionales y no relacionales. Este ciclo también incluye el aprendizaje de herramientas y lenguajes populares como Python, JavaScript, y SQL. Además, se ha dado especial énfasis al desarrollo de aplicaciones.

Tecnologo

Tecnólogo en Desarrollo de Software Universidad del Valle, Palmira (2022-2024) Formación enfocada en el diseño, desarrollo y despliegue de soluciones tecnológicas, con competencias clave en: Desarrollo de Aplicaciones: Uso de lenguajes como Java, Python y JavaScript con frameworks como Spring Boot y React. Bases de Datos: Diseño y manejo de bases relacionales (PostgreSQL, MySQL) y no relacionales (MongoDB). Gestión de Proyectos: Aplicación de metodologías ágiles (SCRUM) para planificación y entrega eficiente. Análisis de Sistemas: Modelado y propuesta de soluciones alineadas a objetivos empresariales.

BootCapm

Bootcamp en Desarrollo Web Full Stack Talento Tech Online, Cali (2024) Formación intensiva centrada en JavaScript y su ecosistema para el desarrollo de aplicaciones Full Stack. Adquirí habilidades prácticas en: Frontend: Creación de interfaces dinámicas y modernas usando React. Backend: Desarrollo de APIs robustas con Node.js y Express. Bases de Datos: Integración con sistemas relacionales (MySQL) y no relacionales (MongoDB). Proyectos Integrales: Implementación de aplicaciones completas, gestionando tanto el frontend como el backend. Control de Versiones: Uso de Git y GitHub para colaboración en equipo.
